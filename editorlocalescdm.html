Importar desde Google Sheets

Arrastra el CSV de Google Sheets aquí

O haz clic para seleccionar el archivo .csv exportado de Google Sheets (UTF-8)

Columnas esperadas (separador ;):
Nº local ; Nombre Negocio ; Categoría ; Nombre Negocio ; Enlace web ; Telefono ; email u otros contactos ; redes sociales ; logo? ; fotos? ; Copy Español ; Copy Inglés

Comportamiento: Se actualizan los campos de texto de cada local. Las rutas de medios (MEDIA_BY_LOCAL) no se tocan. Los locales del CSV que no existan en el HTML se añaden. Los existentes se actualizan.
